# Pomodoro Timer

A desktop productivity timer built with Python that uses the 
Pomodoro Technique to help you stay focused and manage work sessions.

## Technologies Used
- Python 3
- Tkinter (built-in Python GUI library)
- Math (built-in Python library)

## How to Run It
1. Make sure you have Python 3 installed
2. Download ALL these files into the same folder:
   - main.py
   - tomato.png
3. Open your terminal in that folder
4. Run: python main.py

## How to Use It
- Click **Start** to begin a 25-minute work session
- Timer automatically switches between work and break sessions
- Short break (5 mins) after every work session
- Long break (20 mins) after every 4 work sessions
- Checkmarks (✔) track your completed work sessions
- Click **Reset** to start over

## Features
- Countdown timer displayed on a tomato graphic
- Automatically alternates between work and break sessions
- Tracks and displays completed work sessions with checkmarks
- Color-coded labels (Green = Work, Pink = Short Break, Red = Long Break)
- Clean GUI built with Tkinter
